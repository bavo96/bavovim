# bavovim
My custom vim's settings. Currently support Linux and MacOS.
![screenshot](./bavovim.png)

## 1. Prerequisites
- See [prerequisite.md](./docs/prerequisite.md)
## 2. Installation 
### Run the following command:
```bash
bash install.sh -p -n
```
### If you only want to update local neovim:
```bash
bash install.sh -n
```
